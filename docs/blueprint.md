# **App Name**: Classroom Insight

## Core Features:

- Classroom Data Management: Enable admins to add, edit, and delete classroom details including capacity, size, availability status, and current usage using Firestore.
- Real-time Availability Updates: Use Firestore real-time updates to instantly reflect classroom availability changes on the website.
- User Authentication and Roles: Implement Firebase Authentication to manage admin, faculty, and student roles with appropriate access levels. Admin-only access for editing data; view-only access for students and faculty.
- Classroom Search and Filters: Allow users to search classrooms by room number and filter by capacity, building, and availability.
- Availability Forecasting: Integrate AI to predict future classroom availability based on historical usage patterns as a tool for admins when assigning rooms or events.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Very light lavender (#EDE7F6).
- Accent color: Deep violet (#673AB7), for focus and contrast in interactive elements.
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral aesthetic suitable for a professional dashboard
- Use minimalist, clear icons for navigation and status indicators.
- Clean, card-based layout for classroom listings to ensure readability and easy scanning of information.
- Subtle transition animations for real-time updates.
